\subsubsection*{Remarque}
Dans ce chapitre, l'intérêt sera pratiquement toujours \underline{composé}.

\subsubsection*{Quelques définitions...}
\begin{description}
	\item[Rentes (\emph{annuities})]{série de versements périodiques.}
	\item[Rente contingente]{conditionnel à la survie ou que l'entreprise 		reste ouverte.}
	\item[Rente certaine]{C'est sûr à 100\% qu'on recevra les versements 	(\emph{ça sera toujours le cas dans le cours ACT-1001})}
\end{description}	

\subsubsection{\hl{Rappel super important}}
La propriétés des sommes géométriques, \underline{TRÈS IMPORTANT} pour les annuités!
\begin{empheq} [box=\fcolorbox{black}{green}]{equation}
\sum_{j=m}^n ar^j = ar^m \left(\frac{1-r^{n-m+1}}{1-r} \right)
\end{empheq}

\subsubsection{Exemple 2.1}
On reçoit des versements périodiques de 30\$/mois et on est capable d'obtenir un taux d'intérêt nominal annuel composé mensuellement ($i^{(12)}$) de 9\%. Quelle est la valeur accumulée au moment du 140\up{e} versement?
% ------
\p
Tout d'abord, il est important de se faire un graphique : 
\begin{center}   % INSERTION D'UNE IMAGE POUR AIDER À comprendre
\includegraphics[width = 10cm, height = 4cm]{illustrations/ex2_1_chap2.png}
\end{center}
% ------
Ensuite, on convertit notre taux d'intérêt pour le mettre sur la même fréquence que nos versements : 

\begin{gather*}
\frac{i^{(12)}}{12} = \frac{0,09}{12} = 0,75\% \text{ par mois}
\end{gather*}

Après, on doit trouver la valeur accumulée de cette annuité (série de versements). Il s'agît d'une suite de versements qui peut être généralisée comme suit : 

\begin{gather*}
30(1+0,0075)^{139} + 30(1+0,0075)^{138} + ... + 30(1+0,0075)^{2} + 30(1+0,0075)^{1} + 30
\end{gather*}

Avec la propriété des sommes géométriques, on est capable de résoudre rapidement cette suite : 

\begin{gather*}
\sum_{m=1}^{139} 30(1+0,0075)^m = 30 \left(\frac{1- (1+0,0075)^{140}}{1-(1+0,0075)} \right) = 7385,91\$
\end{gather*}
